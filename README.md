Custom QR Code Generator & Scanner

A responsive web application that allows users to generate and scan customizable QR codes with ease. It supports real-time QR creation, color customization, logo insertion, and QR code scanning directly through the device camera.

Tech Stack

HTML5

CSS3

JavaScript

Features

Real-time QR generation
Instantly creates QR codes as the user types.

Logo and color customization
Allows changing QR colors and adding a custom logo at the center.

QR code scanning
Uses the device camera to read QR codes.

Responsive UI
Works smoothly across mobile, tablet, and desktop screens.
